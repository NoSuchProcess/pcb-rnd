pcb-rnd - indentation

As of r1022 all old code is converted using indent(1) with the following command line: 
indent --line-length128 -brs -br -nce --tab-size2 -ut -npsl  -npcs -hnl $*

Contributors are kindly asked to follow this style or run the above commandline on their code before sending a patch or commiting to svn.
