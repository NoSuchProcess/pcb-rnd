8.1. gsch2pcb-rnd

Gsch2pcb-rnd was a standalone utility that could extract netlists and element data from a schematics produced by gschem. It was the old glue between gschem and pcb-rnd, doing forward annotation. It has been replaced by the import schematics feature. The latest version of gsch2pcb-rnd is still accessible as a separate project but should not be used.
