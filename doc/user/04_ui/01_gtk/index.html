Menus

menus for menu command picking, perform standard file operations and specific layout operations The menus provide standard file operations as well as access to prefs and core commands (actions) available in pcb-rnd menus:
  • File - standard file operations -pcb-rnd import/export, and preferences access
  • Edit - a little of everything, undos, buffer ops, selection, edit name and route styles,
  • View - main view grid control and attribute visibiility
  • Settings - lots of things
  • Select - selection and edit selection control
  • Buffer - edit, select, place, transform buffer contents
  • Connects - connection operations and DRC
  • Plugins - Manage plugins
  • Info - Generate reports and get keybindings
  • Window - open a auxiliary window (dialog)
