DStyle = 0 means no thermal.
DStyle = 1 has diagonal fingers with sharp edges.

DStyle = 3 is a solid connection to the plane.Style = 4 has diagonal fingers with rounded edges.
DStyle = 5 has horizontal and vertical fingers with rounded edges.
