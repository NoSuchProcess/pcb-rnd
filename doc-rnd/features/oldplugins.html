pcb-rnd - the [oldplugins]

In this effort I imported a set of old PCB plugins to be core plugins of pcb-rnd. This will make it easier to keep them up to date, to defeat the bitrot effect. The plugins that got imported so far:
  • autocrop
  • boardflip
  • distalign
  • distaligntext

Plugins that won't be imported:

  • findelement - [query] takes care of that
  • findrat - [query] will take care of that
  • join-found - not clear what it would do
  • lockelements - [query] can select elements then SetFlag(selected) should be able to lock the,
  • sedrename - [query] will take care of that
  • upth2pth - [query] will take care of that

save/load and compatibility

Not affected.
